介绍

Protocol Buffers 是Google开源的一种轻便高效的结构化数据存储格式，可以用于结构化数据序列化。它很适合做数据存储或RPC数据交换格式。可用于通讯协议、数据存储等领域的语言无关、平台无关、可扩展的序列化结构数据格式。

Protocol Buffers可以分为两块内容：

  • DSL编写的proto文件，然后使用protoc编译所需要的语言源文件。
  • Protocol Buffers相关语言的依赖库。
